# NegotiationPRO
Тренажер Переговорных навыков
Negotiator Pro — тренажёр переговорных навыков
Интерактивный веб-тренажёр для развития навыков убеждения, аргументации и ведения сложных переговоров.

О проекте
Negotiator Pro — это образовательное веб-приложение, которое помогает пользователям развивать ключевые переговорные компетенции через диагностику, обучение и практические симуляции. Тренажёр построен на реальных психологических техниках и подходах, используемых профессиональными переговорщиками.

Основные возможности
1. Диагностика навыков
10 реалистичных переговорных ситуаций

Оценка по 5 ключевым компетенциям:

Эмпатия

Ассертивность

Стратегическое мышление

Работа с возражениями

Аргументация

Визуализация результатов в виде профиля навыков

Автоматическое определение зон роста

2. Обучающие модули
Теоретические блоки по каждой компетенции

Авторские методики от экспертов (Крис Восс, Роджер Фишер, Карл Роджерс и др.)

Конкретные техники с примерами из реальной жизни:

Активное слушание и парафраз

Зеркалирование (mirroring)

Маркировка эмоций (labeling)

Я-сообщения

Техника «заезженная пластинка»

BATNA (альтернатива переговорному соглашению)

SPIN-вопросы

Язык выгоды (FAB)

И многие другие

3. Интерактивные симуляции
Четыре реалистичных сценария с диалоговой системой:

Сценарий	Сложность	Описание
Повышение зарплаты	Средний	Переговоры с боссом о деньгах после перевыполнения плана
Продажа CRM	Сложный	Убедить консервативного директора купить программное обеспечение
Срыв сроков	Сложный	Компромисс с подрядчиком, который не доставил товар
Аренда квартиры	Лёгкий	Обсуждение повышения арендной платы с хозяином

Финальный разбор полётов с подробным анализом каждого хода

4. Система оценки

Как использовать
Пройдите диагностику — ответьте на 10 вопросов о том, как бы вы поступили в разных переговорных ситуациях.

Изучите результаты — узнайте свои сильные стороны и зоны роста.

Изучите теорию — ознакомьтесь с техниками по компетенциям, которые нужно подтянуть.

Попрактикуйтесь — выберите любой сценарий и пройдите диалог с виртуальным оппонентом.

Проанализируйте — после симуляции получите детальный разбор каждого вашего ответа.

HR-специалисты при проведении интервью и обсуждении условий

Предприниматели на старте переговоров с инвесторами и поставщиками

Все, кто хочет улучшить свои коммуникативные навыки
